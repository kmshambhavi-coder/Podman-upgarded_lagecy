
# CountryFlags

Country Flags includes a list of all the countries and their associated flags. You can use this integration to present a country's  flag as a base64.

Python Version - 3

## Actions
#### Get Country Flag
Gets a country's flag from a list of flags and presents it as a base64
Timeout - 600 Seconds


|Name|Description|IsMandatory|Type|DefaultValue|
|----|-----------|-----------|----|------------|
|Country Code Two Digit|Enter the 2 digits of a given country (for example, us)|False|String||
|Country Code From Entity Field|Insert a given field related to a specific entity that has been enriched by another product that you would like to a extract a base64 for that country|False|String|None|
